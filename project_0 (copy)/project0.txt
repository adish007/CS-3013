^[[1mDESCRIPTION^[[0m^[[m^M
       The  ^[[1mstrcpy^[[0m()  function copies the string pointed to by ^[[4msrc^[[24m, including the terminating^[[m^M
       null byte ('\0'), to the buffer pointed to by ^[[4mdest^[[24m.  The strings may not overlap,  and^[[m^M
       the  destination string ^[[4mdest^[[24m must be large enough to receive the copy.  ^[[4mBeware^[[24m ^[[4mof^[[24m ^[[4mbufM-bM-^@M-^P^[[24m^[[m^M
       ^[[4mfer^[[24m ^[[4moverruns!^[[24m  (See BUGS.)^[[m^M
^[[m^M
       The ^[[1mstrncpy^[[0m() function is similar, except that at most ^[[4mn^[[24m  bytes  of  ^[[4msrc^[[24m  are  copied.^[[m^M
       ^[[1mWarning^[[0m: If there is no null byte among the first ^[[4mn^[[24m bytes of ^[[4msrc^[[24m, the string placed in^[[m^M
       ^[[4mdest^[[24m will not be null-terminated.^[[m^M
^[[m^M
       If the length of ^[[4msrc^[[24m is less than ^[[4mn^[[24m, ^[[1mstrncpy^[[0m() writes additional null bytes to ^[[4mdest^[[24m to^[[m^M
       ensure that a total of ^[[4mn^[[24m bytes are written.^[[m^M
^[[m^M
       A simple implementation of ^[[1mstrncpy^[[0m() might be:^[[m^M
^[[m^M
           char *^[[m^M
